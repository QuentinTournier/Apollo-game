Projet Apollo-game réalisé par Quentin Tournier :

Ce projet est un client de l'API IGDB

Ce projet est réalisé en Java, avec Spring et tourne sur un serveur Tomcat

Pour lancer le projet:
-Il faut avoir java installé,
-Avoir configuré un serveur Tomcat sur la machine locale
-Avoir ouvert le code avec un IDE Java (IntelliJ (celui que j'ai utilisé) ou NetBeans ou autre
-Lancer le projet (Si demandé, la classe principale est "Application")
-Aller à l'url "localhost:8080" sur votre navigateur préféré
